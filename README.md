# Jones Barber Shop

Community barbershop website for Jones Barber Shop — Joneswood District. Sharp cuts, real conversations, since 1998.

## Live Site

> Add GitHub Pages URL here once enabled.

## Stack

- Vanilla HTML, CSS, JavaScript — single file, no build step
- Static site — no backend, no database, no framework
- Deployment target: GitHub Pages

## Sections

Hero · About · Services · Barbers · Gallery · Pricing · Booking · Location · Reviews · Footer

## Local Development

Open `index.html` directly in a browser, or use Live Server (VS Code, port 5501).
