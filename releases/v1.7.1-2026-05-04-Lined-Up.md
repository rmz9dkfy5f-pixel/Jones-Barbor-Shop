# v1.7.1 - 2026-05-04 - Lined Up

> **Lined Up** — precision patch. Corrects two visual issues with the Book Now button in the fixed navigation bar: button position in the header layout and text centering inside the pill.

---

## What's Changed

### Fixed
- `.header-inner` — switched from `flex` + `justify-content: space-between` to a 3-column CSS grid (`auto 1fr auto`) so the nav anchors to the center column and the CTA sits right-aligned without excessive gap
- `.nav-cta` — reduced `gap` from `1rem` to `0.75rem` to tighten the walk-in label and button grouping
- `.btn-primary` — reduced `letter-spacing` from `0.16em` → `0.06em` → `0` to eliminate the invisible trailing-space offset that made button text appear left-shifted inside the pill
- `.header-inner nav` — added `display: flex; justify-content: center` so nav links sit centered in the grid's middle column

> 1 file changed across 4 commits — `index.html` only
