# v1.11.0 - 2026-06-14 - Quality Gates

> **Quality Gates** — installs the Project Starter Kit v3.3 quality gate layer (Group 1 of the v3.3 migration). Six new root-level and `ai/agents/` files define done criteria, change control rules, repo health baseline, rollback procedures, risk register, and agent review gates. No existing files were modified.

---

## What's Changed

### Added
- `DONE_CRITERIA.md` — defines what "done" means for every slice type (feature, debug, design, migration)
- `CHANGE_CONTROL.md` — defines what AI agents may edit freely, what requires approval, and what is never automatic
- `REPO_HEALTH_CHECK.md` — baseline snapshot of repo stack, important files, risks, and verification status
- `ROLLBACK_PLAN.md` — escape path template for migrations and risky changes; pre-filled for this migration
- `PROJECT_RISK_REGISTER.md` — open and closed risk log; seeded with known Jones Barber Shop risks
- `ai/agents/AGENT_REVIEW_GATES.md` — mandatory gate rules for sub-agent invocation by change type

> 6 files added — no existing files modified
