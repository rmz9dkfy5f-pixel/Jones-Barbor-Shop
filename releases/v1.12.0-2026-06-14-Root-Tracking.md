- `BACKLOG.md` — work queue organized by Build Now / Build Next / Build Later / Maybe / Do Not Build

- `AGENTS.md` — agent instructions including commit rules, required files, and sub-agent workflow
- `PROGRESS_NOTES.md` — historical progress log; seeded with v1.10.0 and v1.11.x session notes
- `DECISION_LOG.md` — key decision log; seeded with 4 active decisions
